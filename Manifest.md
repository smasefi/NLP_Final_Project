## Description
The corpus used in this project consists of 9 works by 5 authors - totaling 63 chapters of written work. There are 4 ideological political schools of thought represented here. 

1. Utopian Socialism - Charles Fourier
2. Marxism - Karl Marx and Friedrich Engels
3. Democratic Socialism - Rosa Luxemburg
4. Maoism - Mao Zedong

The point of incliuding this many contrasting schools of thought was to understand how they relate or differ from each other. Each of these thinkers are categorized under the same huge umbrella, so this project seeks to investigate how they relate or differ.

## Format

Source Files: plain text (.txt), one file per chapter that are organized subdirectories.

Internal file structure: Each .txt file uses a lightweight XML-like format with source url, page title, chapter title, paragraph id tags from the initial HTML scraping before being converted into a .txt

The corpus/f2 contains .csv's of DOC.csv, LIB.csv, TOKEN.csv and VOCAB.csv

The corpus/f4 contains DTCM.csv, TFIDF.csv, VOCAB.csv - the separate tables to investigate these values and an appended VOCAB.csv

The corpus/f5 contains DOC_SENTIMENT.csv, LDA_DOC_TOPIC.csv, PCA_DCM_chaps.csv, PCA_TCM_chaps.csv, VOCAB_F5.csv (appended with these tables)

**Jupyter Notebooks (run in order):**

1. `01_fetchdata.ipynb` — scrapes and saves source HTML from the Marxists Internet Archive
2. `02_cleandata.ipynb` — tokenizes, annotates, and produces F2 tables (LIB, DOC, TOKEN, VOCAB)
3. `03_bow_tfidf.ipynb` — builds BOW, DTCM, and TFIDF matrix (F4)
4. `04_pca.ipynb` — principal component analysis, produces DCM and loadings (F5)
5. `05_lda.ipynb` — latent Dirichlet allocation, produces topic tables (F5)
6. `06_word2vec.ipynb` — trains word embeddings and adds to VOCAB_F5 (F5)
7. `07_sentiment_analysis.ipynb` — NRC and VADER sentiment analysis (F5)
8. `08_hierarchial.ipynb` — hierarchical clustering and dendrograms

All notebooks can be executed sequentially using `run_pipeline.py`. If the corpus files already exist in `corpus/books/`, `01_fetchdata.ipynb` can be skipped.
